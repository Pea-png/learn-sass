## learn-sass
Project Description :
- Hands-on Sass Practice: Learn and implement Sass features like variables, nesting, and mixins.
- Component-based Styling: Organize styles in modular SCSS files to follow a scalable architecture.
- Variables & Theming: Implement color palettes, typography scales, and spacing systems using Sass variables and maps.
- Skills Gained: Strengthens practical CSS workflow, improves maintainability, and prepares for real-world projects.
